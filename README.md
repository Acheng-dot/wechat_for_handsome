## 环境需求

PHP >= 7.1

PHP cURL 扩展

PHP OpenSSL 扩展

PHP SimpleXML 扩展

PHP fileinfo 扩展

PHP PDO_MYSQL 扩展

## 食用方法
1、拷贝项目到你的服务器
 
2、打开 网址/install.php 安装

3、在公众号后台服务器配置填写服务器地址为 项目所在网址/server.php 并启用服务器配置

4、公众号发送绑定，点击链接填写相关信息进行绑定
 
5、配置完成，尽情使用吧！使用方法同handsome官方时光机发送公众号
